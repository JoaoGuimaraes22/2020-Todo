# Part 1 | Project Setup

## Steps

### Set up project

- Make project 2D mode;
- Go to your icon.png;
- Go to the "Import" tab;
- De-select "Filter", by clicking the checkbox;
- Reimport;
- Go to "Preset";
- Click "2D Pixel";
- Reimport;
- On "Preset", click "Set as default for "Texture""
- Reimport;
- Now import all your sprites, images, etc.
